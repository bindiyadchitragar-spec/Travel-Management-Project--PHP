# Travel Management System

This is a Travel Management System built with PHP and MySQL. It allows users to book and manage travel packages, make inquiries, report issues, and more. The project uses a MySQL database to store user data, bookings, inquiries, issues, and packages.

## Features

- User registration and login.
- Booking travel packages.
- View available travel packages.
- Report issues and track their status.
- Manage bookings and their statuses.

## Prerequisites

To run this project locally, you need:

- **PHP 8.x** or later
- **MySQL** (or MariaDB)
- **phpMyAdmin** (optional, for managing the database)
- **A web server** like Apache or Nginx (e.g., XAMPP or WAMP for Windows)
- HERE I USED XAMMP SERVER

Clone this repository to your local machine using:
'''bash

STEP 2:

Set up the MySQL Database
Open phpMyAdmin or MySQL Workbench and create a new database called tourism_management.
Import the provided SQL dump into the database.
Open phpMyAdmin.
Select the tourism_management database.
Click on the Import tab.
Choose the SQL file provided (database_dump.sql) and click Go.
The SQL file contains the necessary tables and sample data for the bookings, inquiries, issues, packages, and users tables.

STEP 3:

Run the Project
Ensure that your web server (Apache or Nginx) is running.
Place the project files in the htdocs (for XAMPP) or your web server's root directory.
Open your web browser and navigate to:
http://localhost/travel-management
